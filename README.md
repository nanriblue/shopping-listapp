# 쇼핑 리스트 앱

간단한 쇼핑 리스트 웹 애플리케이션입니다. HTML, CSS, JavaScript로 구성되어 있으며, LocalStorage를 사용하여 데이터를 저장합니다.

## 기능

- 쇼핑 아이템 추가 / 삭제
- 체크(완료) 표시
- LocalStorage 기반 데이터 저장 (새로고침해도 유지)
- 남은 아이템 수 표시

## 실행 방법

1. 저장소를 클론합니다:
   ```bash
   git clone git@github.com:nanriblue/shopping-listapp.git
   ```
2. `public/index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.
